Overlow es una propiedad de html que tiene el fin de corregir contenido excedente (que excede) en un elemento a nivel de bloque. Ejemplo un cuadro de color.


VISIBLE EJEMPLO

El contenido al que se le asigna el valor visible se deja ver aun si se encuentra fuera del cuadro.

SCROLL EJEMPLO

El contenido al que se le asigna el valor scroll es escondido se esconde pero se puede acceder a él gracias a una barra de scroll. De esta manera se mantiene dentro del cuadro y podemos acceder a él.


AUTO EJEMPLO

El contenido al que se le asigna el valor auto será visible mediante una barra de scroll solo si la necesita. Si el contenido no sobrepasa los límites del cuadro la barra scroll no será agregada.
